## Einleitung Das LOM-Applikationsprofil des Open-Edu-Hubs wird hier dokumentiert.
## Allgemeine Beschreibung
## Das Element general Grundlegende Informationen, die das Lernobjekt als Ganzes beschreiben.
Mehrwertigkeit:
Es MUSS genau 1 Mal vorkommen.
Elemente:
identifier
title
language
description
keyword
coverage
structure
aggregationlevel
## Das Element identifier Das Element bezeichnet eine innerhalb des Repositoriums eindeutige Identifizierung. Es hat Werte vom Typ `String`, die keiner Einschränkung unterliegen. Bei Anfragen an den OAI-Server eines Repositoriums ist der Wert dieses Elements das bestimmende Identifikationsmerkmal für ein Lernobjekt.
Mehrwertigkeit:
Es MUSS genau 1 Mal innerhalb des Elements general vorkommen.
Attribute:
keine
Elemente:
keine
